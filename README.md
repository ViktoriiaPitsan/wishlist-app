## üöÄ Features

- **Full CRUD Operations**: Create, Read, Update, and Delete wishes seamlessly.
- **Dynamic Dashboard**: A grid-based layout with interactive wish cards.
- **Filtering**: Sort wishes by date (newest/oldest) and price (high/low).
- **Fully Responsive**: Optimized for a seamless experience across all devices, from mobile phones (min 320px) to large desktop monitors
- **Interactive UI**: Smooth hover effects and transitions for better user engagement.

## üì¶ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ViktoriiaPitsan/article-filter-app.git
   ```

2. **Enter the directory**:
   ```bash
   cd wishlist
   ```
3. **Install dependencies:**:
   ```bash
   npm install
   ```
4. **Run the Mock Back-end:**:
   ```bash
   npm run server
   ```

4. **Run the app**:
   ```bash
   npm run dev
   ```
